# N-gram generator
Ipsum lorem and what not. 


## Usage
ngram.py extracts n-grams from textfiles. If no files are given stdin is used.

`python3 ngram.py n [file]`

lorem.py uses n-grams generated by ngram.py to create random strings
with (approximately) the same frequency distribution of n-grams as the
original input. The number of characters to generate must be given as an argument.
Input is from the named file argument or stdin if no file is given.

`python3 lorem.py n [file]`

### Example:

 `python3 ngram.py 4 text/english.txt | python3 lorem.py 1000`
